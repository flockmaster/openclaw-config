# HEARTBEAT.md - 数学老师定期任务

## 每日推题（目标：下午 5 点）

每次心跳检查以下内容：

1. **检查当前时间**：如果是下午 4:30 ~ 5:30 之间，执行推题流程
2. **检查今日是否已推题**：读取 `data/daily-records/`，看今天是否已有推题记录。如果已推过，跳过
3. **生成每日一题**：
   - 读取 `data/progress.json`，找到当前 `in_progress` 的单元（没有则取第一个 `not_started` 的）
   - 读取 `data/mistake-book.json`，检查是否有到期复习的错题（`next_review_date <= 今天` 且 `status == "learning"`）
   - 读取 `data/adaptive-difficulty.json`，按当前难度等级出题
   - **优先推复习题**，没有复习题则按当前进度出新题
4. **推送到飞书**：发送消息给汐汐，格式友好，附带每日数学小知识
5. **记录**：在 `data/daily-records/` 中记录今日推题

### 推题消息模板

```
汐汐～今天的数学时间到啦！🎯

📝 今日一题（{难度等级}）：
{题目内容}

💡 你知道吗？{数学小知识}

加油哦！做完了告诉老师答案～
```

## 非推题时段

如果不在推题时段，回复 HEARTBEAT_OK。
