# 学习进度与弱项攻克

## 汐汐的弱项

### 1. 应用题读不懂
**表现**：题目太长不知道问什么，不知道用什么方法

**攻克方法**：
- 🏃 **三步法读题**
  - 第一步：找"谁"（谁在做事）
  - 第二步：找"做了什么"（动作/变化）
  - 第三步：找"问什么"（最后求什么）
  
- 📝 **画关键词**
  - 多了、少了、共、还剩 → 加法
  - 还剩、相差、减少 → 减法
  - 每份、几个几倍 → 乘法
  - 平均分、每个多少 → 除法

### 2. 粗心
**表现**：算对了但写错了、漏题、看错符号

**攻克方法**：
- 🕵️ **验算习惯**
  - 养成每题必验算的习惯
  - 应用题答案代入检查
  
- 📝 **划关键词**
  - 读题时用手指指着每个字
  - 圈出"坑"（如：单位的不同）
  
- 💪 **细心训练**
  - 草稿纸要整洁
  - 抄写要检查

---

## 每日练习

### 应用题专项训练
- 每天至少1道应用题
- 做应用题时必须用"三步法"
- 做完后要口述：这道题告诉我们什么？要求什么？

### 粗心专项训练
- 每次做题前先深呼吸
- 写完检查：数字对不对？单位写了没？符号看清楚了没？
- 草稿纸对折分区，清晰整洁
