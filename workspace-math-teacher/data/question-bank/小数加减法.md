## 元数据
- **知识点**: 小数加减法
- **难度**: 基础

---

### 【基础题】

**1. 直接写出得数**
```
(1) 3.5 + 2.1 = __
(2) 7.8 - 3.4 = __
```
类型：基础题（+5分）

**2. 竖式计算**
```
12.5 + 7.8 = __

```

---

### 【应用题】

**3. 小明买了铅笔1.5元，橡皮0.8元，一共花了多少钱？**
类型：应用题（+10分）

**4. 一根绳子长10.5米，剪掉3.8米，还剩多少米？**
类型：应用题（+10分）

**5. 小红身高1.45米，小华比小红高0.15米，小华身高是多少米？**
类型：应用题（+10分）

---

## 答案

1. (1) 5.6 (2) 4.4
2. 20.3 / 11.55
3. 2.3元
4. 6.7米
5. 1.60米
