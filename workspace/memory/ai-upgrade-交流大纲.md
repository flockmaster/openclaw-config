## 三、多智能体协同系统（8分钟）

**核心价值：从"工具"到"团队"**

### 系统架构
```
┌─────────────────────────────────────────┐
│              主智能体（调度）              │
│         接收需求 → 分解任务 → 派发          │
└─────────────────┬───────────────────────┘
                  │
    ┌─────────────┼─────────────┐
    ▼             ▼             ▼
┌───────┐   ┌───────┐   ┌───────┐
│产品经理│   │前端工程师│   │后端工程师│
│ 专家  │   │  专家  │   │  专家  │
└───────┘   └───────┘   └───────┘
```

### 工作流
需求 → PRD → 原型 → 评审 → 开发

### 核心能力
1. **记忆系统**：上下文接续，不"失忆"
2. **自动请专家**：问题匹配专家，按需调用
3. **SOP固化**：成熟流程可固化为专家Agent

### 愿景
- 企业内部的成熟SOP → 专家Agent
- 内部发布共享
- 形成AI能力积累
