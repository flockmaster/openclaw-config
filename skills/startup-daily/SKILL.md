# startup-daily - 每日创业项目精选播报

> 抓取全球创业项目和融资动态，过滤非软件相关项目，生成语音播报。

## 触发条件
- 定时任务调用（每天凌晨 5 点）
- 用户主动询问"今日创业项目"

## 工作流程

1. **搜索**：用 web_search 搜索今日创业项目和融资动态
   - 关键词：'独立开发 融资 2026'、'SaaS 创业 2026'、'AI 初创 融资'、'Product Hunt 今日热门'
   
2. **过滤**：排除非软件/互联网项目
   - 排除关键词：养猪、农业、种植、养殖、农场、畜牧、渔业、农民
   - 保留：软件、AI、互联网、SaaS、App、开发者工具、技术平台

3. **整理**：精选 3-5 个项目
   - 项目名称
   - 一句话介绍
   - 亮点/融资额
   - 原文链接

4. **文案**：编写口播文案（口语化，300-500 字）

5. **语音**：调用 tts 工具生成语音

6. **发送**：用 message 工具发送飞书给用户

## 使用方法

### 定时任务调用
```
CRON_TRIGGER: 每日创业项目精选（软件/互联网方向）。
立即执行 startup-daily skill 完整流程。
```

### 手动测试
```
执行 startup-daily skill，发送测试播报到飞书。
```

## 注意事项

- 只要软件/互联网/AI 相关的创业项目
- 不要养猪、农业、传统行业
- 口播文案要口语化，像朋友聊天一样
- 每个项目附上原文链接
- TTS 超时则降级为纯文字播报
